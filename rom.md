# ROM(Read Only Memory)
- ROM
  - Mask ROM
  - PROM
    - OTPROM
    - EPROM
      - EEPROM
        - Flash Memory

## ROM
- 基本読み出し専用の記憶装置
- 種類によっては全消去、書き込み、追記が可能

## Mask ROM
書き換えが不可能

## PROM(Programmable ROM)
書き込みが可能

## OTPROM(One Time PROM)
1度のみ書き込み可能

## EPROM(Erasable PROM)
書き込みと消去が可能

## EEPROM(Electically EPROM)
電圧をかけることで何度も書き込みや消去が可能

## Flash Memory
ブロック単位での消去や書き込みが可能
